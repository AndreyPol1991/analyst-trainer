# Бюро анализа на Бейкер-стрит

Игровой тренажёр по лекции «Этапы разработки ПО» для будущих системных аналитиков.
Детективная стилизация по мотивам Шерлока Холмса: 9 дел, 106 интерактивных улик-кейсов.

**Темы:** жизненный цикл ПО · участники команды · роль аналитика · требования (ФТ/НФТ, MoSCoW) · техники сбора требований · методологии (Waterfall, Scrum, Kanban) · use case и user story.

**Механики:** кейсы с выбором решения, хронологии, сопоставления, сортировка улик,
конструктор user story, допросы свидетелей-стейкхолдеров в формате чата,
экспертиза ТЗ («найди подозрительную строку»), блиц-допрос с таймером.
XP, серии, звания, разбор ошибок. Прогресс хранится в localStorage браузера.

Один файл `index.html` без зависимостей — работает офлайн, тёмная и светлая темы.
